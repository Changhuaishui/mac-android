# 工作边界

## 允许

- 在本项目目录中创建和修改项目文档。
- 在明确任务卡范围内创建 Mac、Android、协议代码。
- 执行只读检查命令。
- 执行用户授权的构建、测试、安装和运行命令。
- 记录失败、未验证项和下一步。

## 禁止

- 擅自上传代码、日志、截图或屏幕内容到云服务。
- 擅自引入账号系统、遥测或远程公网访问。
- 擅自提交、推送、发布或安装系统扩展。
- 擅自绕过 macOS 权限提示。
- 把私有 API 路线包装成稳定公开方案。
- 为了追求自动化而引入复杂平台。

## 需要再次确认

- 使用 macOS 私有 API。
- 开始 DriverKit 或系统扩展开发。
- 安装依赖、SDK、驱动或证书。
- 执行会修改系统权限、网络配置或设备设置的命令。
- 连接真实 Android 设备进行安装或调试。
- 删除、覆盖或迁移已有目录。

## 状态定义

| 状态 | 含义 |
|---|---|
| 待确认 | 需求、范围或授权还不明确 |
| 可实施 | 范围和完成定义明确 |
| 实施中 | 主 Agent 正在执行 |
| 代码完成 | 改动完成，但验证未全部完成 |
| 验证完成 | 已完成约定验证 |
| 待人工验收 | 技术验证完成，等待用户判断 |
| 已完成 | 用户确认结束 |
| 阻塞 | 缺少环境、权限、数据或用户选择 |
