## 4. 当前事实

- 已确认：M1 使用 TCP 长连接、H.264、Protocol v0。
- 已确认：推荐帧头为 `magic/version/type/sequence/timestamp_ns/flags/payload_len/payload`。
- 当前假设：控制类消息 payload 使用 UTF-8 JSON，视频帧 payload 使用 H.264 Annex B byte stream。
- 尚未验证：Mac 端 VideoToolbox 输出转换、Android 端 MediaCodec 真机解码。
